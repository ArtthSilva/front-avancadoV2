## Installation and Configuration

### Prerequisites

Make sure you have installed:

- Node.js (version 23.3.0 or higher)
- npm or yarn

### Installation Steps

1. **Clone this repository:**

   ```bash
   git clone https://github.com/ArtthSilva/front-avancadoV2.git
   ```

2. **Access the project directory:**

   ```bash
   cd front-avancadoV2
   ```

3. **Install dependencies:**

   ```bash
   npm install
   ```

   or

   ```bash
   yarn install
   ```

## How to Use

### Start the Development Server

Run the command:

```bash
npm run dev
```

The project will start and can be accessed in the browser at [http://localhost:5173](http://localhost:5173).
